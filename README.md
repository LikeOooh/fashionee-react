# Фронт проекта Fashionee

## Установка пакетов

Из корня проекта выполнить
```bash
npm install
```
### Разметка директорий

* dist
    * → Mark Directory as Resource Root
* src
    * → Mark Directory as Sources Root

## Запуск проекта

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
